# Quick Start Guide

## Prerequisites

- Node.js (v14+)
- Python (v3.8+)

## Setup & Run

**1. Start Backend**

```bash
cd backend
pip install fastapi uvicorn
python main.py
```

Backend runs at: http://localhost:8000

**2. Start Frontend** (in a new terminal)

```bash
cd frontend
npm install
npm start
```

Frontend runs at: http://localhost:3000

---

**That's it!** Open http://localhost:3000 to use the app.
